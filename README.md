# Exercício pertencente a disciplina de Engenharia de Software III

## Design Patterns

### Observer

Em uma rede de publicação de conteúdo, toda vez que você se inscreve (“Ativa o sininho”), você recebe uma mensagem a  cada vez que um novo conteúdo é postado. 

Considerando a estrutura Observer, em que o inscrito é o observador e o criador de conteúdo é o observado, construir uma solução em Java que simule uma nova postagem de um criador de conteúdo e 3 inscritos em um List de Inscritos.
